# 🔄 SkillSwap — Student Skill Exchange Platform

A full-stack MERN application where students can list skills they can teach, find peers, and request skill swaps.

## 🌐 Live Demo
[Coming soon after deployment]

## ✨ Features
- 🔐 JWT Authentication (Register/Login)
- 👤 Student Profiles with skills listing
- 🔍 Browse & Search students by skill
- 🤝 Send/Accept/Reject Swap Requests
- 💬 Real-time Chat (Socket.io)
- ⭐ Ratings & Reviews after swaps
- 📱 Responsive UI with Tailwind CSS

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React.js, Vite, Tailwind CSS |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Auth | JWT, bcryptjs |
| Real-time | Socket.io |
| Deployment | Vercel (FE) + Render (BE) |

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Environment Variables
Create `backend/.env`:
```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
```
